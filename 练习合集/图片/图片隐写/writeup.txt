# binwalk分析图片，发现其中还有一个elf
binwalk fe02.jpg

DECIMAL       HEXADECIMAL     DESCRIPTION
--------------------------------------------------------------------------------
0		0x0		JPEG image data, JFIF standard 1.01
113089		0x1B9C1		ELF, 64-bit LSB shared object, AMD x86-64, version 1 (SYSV)

使用dd命令分离
dd if=fe02.jpg of=fe02.elf skip=113089 bs=1
# if指定输入文件，of指定输出文件，skip跳过开头，bs设置每次读写块的大小为1字节

得到fe02.elf，运行得到flag
